# NeighbourNet

## Hyperlocal Community Resource Sharing Platform

NeighbourNet is a hyperlocal community resource sharing platform designed for apartment complexes, hostels, and gated communities. The platform enables verified community members to share, borrow, and request rarely used items such as drills, projectors, camping gear, cleaning tools, and other household equipment.

The project aims to reduce unnecessary individual purchases, encourage sustainable resource usage, and strengthen social interaction within modern urban communities.

---

## Problem Statement

People living in apartment complexes or hostels often buy expensive, rarely used items that remain idle most of the time, while others nearby purchase the same products separately. There is currently no simple and trusted platform for localized resource sharing within such communities.

NeighbourNet addresses this issue by creating a secure, community-based “Library of Things” system.

---

## Key Features

- Community-based user onboarding
- Restricted access for verified residents
- Item listing and browsing system
- Booking and rental workflow
- Refundable deposit simulation
- Request item feature
- Community replies for requested items
- User profile section
- Booking management
- Responsive modern UI
- Local storage-based data handling

- Limited to localized communities
